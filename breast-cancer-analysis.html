When complexity parameter uses the cp of the lowest xerror (cp=0.01), the pruned decision tree looks exactly the same as the original decision tree, therefore resulting the same AUC.

As the complexity parameter increases from 0.01 to 0.9, it simplifies the pruned decision trees and the AUC decreases from 0.94 to 0.5.
